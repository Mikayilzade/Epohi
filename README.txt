ЭПОХИ v1.4-alpha — ОФЛАЙН-СТРАТЕГИЯ

Что нового:
- первые цивилизации-соперники: 0–2 при создании партии;
- соперники исследуют мир, строят юнитов, улучшают клетки и основывают города;
- отдельный туман войны, ресурсы, армии, отношения и журнал решений ИИ;
- простые отношения: неизвестен, нейтрален, война, с защитным периодом до 20 хода;
- панель «Цивилизации» и экран «Летопись» в игровом меню;
- журнал крупных событий сохраняется вместе с campaigns, manual saves, quicksave и autosaves;
- обновлена вики, версия и cache service worker.

Ограничения первой версии:
- дипломатия сведена к войне и простому миру;
- два соперника используют одну базовую логику ИИ с разными именами, цветами и символами;
- форпосты и города ИИ упрощены относительно будущих версий;
- полноценного проигрывателя повторов пока нет, только летопись событий.

Установка на GitHub Pages:
1. Загрузите все файлы из этой папки в корень репозитория.
2. Не загружайте только ZIP — его нужно предварительно распаковать.
3. В GitHub откройте Settings → Pages и выберите Deploy from a branch.
4. Откройте опубликованную ссылку в Safari.
5. Поделиться → На экран «Домой».

При обновлении старой версии замените все файлы, включая sw.js и manifest.webmanifest. Если iPhone показывает старую игру, откройте адрес в Safari и обновите страницу один раз.
